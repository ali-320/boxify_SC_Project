# **App Name**: Boxify

## Core Features:

- Navigation Bar: A responsive navigation bar with links to different sections of the website and the company logo.
- Hero Section: Eye-catching hero section with a compelling headline and a prominent 'Get a Quote' button.
- Featured Products: Display of featured products with image placeholders and short descriptions.
- Company Benefits: A section highlighting key benefits such as 'Free Shipping' or 'Custom Design Options'.
- Footer: A footer section with contact information, copyright notice, and links to important pages.
- Styling: Clean and modern CSS design with a fixed navigation bar, clear hover effects on buttons, and a responsive grid layout for product sections.
- Quote Page: A multi-step 'Request a Quote' form asking for product dimensions, quantity, material, printing options, and user contact information with file upload for artwork.
- Form JavaScript: JavaScript to manage the multi-step form, show/hide steps, and perform basic validation of form fields.

## Style Guidelines:

- Primary color: Saturated blue (#2962FF), evoking reliability and trustworthiness.
- Background color: Light blue (#E0E8FF), subtly tinted, but desaturated.
- Accent color: Purple (#8F29FF), adding a modern, eye-catching contrast.
- Headline font: 'Space Grotesk', sans-serif, for a modern, tech-forward aesthetic. Body font: 'Inter', sans-serif, for clean readability.
- Clean and modern layout with clear sectioning and intuitive navigation. Utilize a grid system for consistency.
- Simple, geometric icons to represent company benefits and product features.
- Subtle animations and transitions on hover for a polished user experience.
